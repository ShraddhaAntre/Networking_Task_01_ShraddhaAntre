## Networking Concepts

### What is an IP Address?
An IP Address is a unique number assigned to a device on a network. It helps devices communicate with each other.

### What is a MAC Address?
A MAC Address is a unique hardware identifier assigned to a network interface card (NIC).

### What is a Default Gateway?
A Default Gateway is the device (usually a router) that forwards traffic from a local network to other networks, including the internet.

### What is DNS?
DNS (Domain Name System) translates domain names such as google.com into IP addresses.

### Public IP vs Private IP

| Public IP | Private IP |
|------------|------------|
| Used on the Internet | Used inside local networks |
| Assigned by ISP | Assigned by Router |
| Globally Unique | Not Globally Unique |
